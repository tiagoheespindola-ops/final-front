# Meu Portfólio Acadêmico

## O Projeto
Este site foi construído para apresentar meu portfólio de desenvolvedor e funcionar como avaliação prática da disciplina. A ideia principal foi construir tudo do zero, aplicando os conceitos fundamentais da web moderna de forma pura.

## Tecnologias Utilizadas
* **HTML5:** Estrutura geral da página usando tags semânticas para melhor organização.
* **CSS3:** Todo o design visual do site, feito de forma responsiva sem o uso de frameworks externos.
* **JavaScript:** Lógica de monitoramento do formulário de contato, validações em tempo real e atualização dinâmica da tela.

## Divisão do Site
O projeto conta com uma página única dividida em quatro partes de fácil acesso:
1. **Início (Home):** Mensagem de introdução e recepção.
2. **Quem Sou (Sobre):** Um breve resumo sobre mim e meus objetivos.
3. **Minha Jornada (Curriculo):** Cards exibindo layouts criados anteriormente.
4. **Fale Comigo (Contato):** Área de contato com tratamento de dados e mensagem de envio concluído.

## Principais Recursos Criados
* Rolagem suave nativa ao navegar pelos links do menu.
* Validação de campos obrigatórios (impedindo o envio de dados incorretos).
* Interação dinâmica ocultando o formulário e exibindo uma mensagem de sucesso após o envio bem-sucedido.
* Log interno simulando o salvamento das informações coletadas no console.
